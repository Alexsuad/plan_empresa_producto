# Regla — Alcance de fase producto

Este repositorio está en fase de producto y arquitectura funcional.

Permitido:

- documentación de producto;
- diagnóstico del repositorio;
- arquitectura funcional;
- roles funcionales;
- gates;
- requisitos de skills;
- estructura de preinformes;
- blueprint funcional.

No permitido sin aprobación explícita:

- backend;
- frontend;
- base de datos;
- agentes ejecutables;
- MCP;
- despliegue;
- automatizaciones productivas;
- cambios destructivos.
